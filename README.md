# 📊 Desafío 1: Análisis de Datos de Alura Store

¡Bienvenido al primer desafío!

En este proyecto ayudarás al **Sr. Juan**, propietario de la cadena **Alura Store**, a tomar una decisión estratégica para su negocio. El Sr. Juan planea iniciar un nuevo emprendimiento y necesita decidir **qué tienda de su cadena debería vender**.

Para ello, analizarás **datos de ventas, rendimiento y reseñas de clientes** correspondientes a las **cuatro tiendas de Alura Store**. El objetivo es identificar **la tienda con menor desempeño** y presentar una **recomendación final basada en datos**.

---

## 🎯 Objetivo del desafío

Realizar un **análisis exploratorio de datos (EDA)** para evaluar el desempeño de cada tienda y determinar cuál es la menos eficiente, utilizando herramientas de análisis y visualización de datos en Python.

---

## 🧠 Lo que practicarás

Durante este desafío pondrás en práctica las siguientes habilidades:

- 📂 **Carga y manipulación de datos** en formato CSV utilizando la librería **Pandas**  
- 📈 **Visualización de datos** con la librería **Matplotlib**  
- 📊 **Análisis de métricas de negocio**, como ingresos, reseñas y rendimiento de ventas  

---

## 📋 Requisitos del proyecto

### 1️⃣ Análisis de datos de las tiendas

Debes analizar diferentes indicadores clave para evaluar el desempeño de cada tienda, incluyendo:

- **Ingresos totales**
- **Categorías de productos más vendidas**
- **Reseñas y calificaciones de clientes**
- **Productos más vendidos**
- **Promedio de envíos**

---

### 2️⃣ Creación de visualizaciones

Para presentar los resultados de manera clara y comprensible, deberás crear **al menos 3 tipos diferentes de gráficos**.

Algunos ejemplos incluyen:

- Gráficos de **barras**
- Gráficos **circulares**
- Gráficos de **dispersión**
- Otros tipos de visualización que faciliten el análisis

---

### 3️⃣ Recomendación final

Una vez finalizado el análisis, deberás redactar una **conclusión basada en los datos**, explicando:

- **Qué tienda debería vender el Sr. Juan**
- **Las razones que respaldan esta decisión**

Tu recomendación debe estar **justificada con los resultados del análisis y las visualizaciones generadas**.
