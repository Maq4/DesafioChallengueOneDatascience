# DesafioChallengueOneDatascience
Desafío de análisis de datos de Alura Store. Se analizan ventas, reseñas de clientes, categorías de productos y rendimiento de cuatro tiendas utilizando Python, Pandas y Matplotlib. El objetivo es identificar la tienda con menor desempeño y presentar una recomendación basada en datos para apoyar una decisión de negocio
